# TLDR Index

Editorial web archive backed by the normalized public dataset in [`GaloisField2718/tldr_news`](https://github.com/GaloisField2718/tldr_news).

```sh
npm ci
npm run data:sync
npm run dev
```

See [docs/DATA_INTEGRATION.md](docs/DATA_INTEGRATION.md) for source pinning, validation, generated artifacts, search architecture, and deployment behavior. The Daily newspaper’s deterministic fallback, validated editorial overlay, direct Worker illustration, and immersive reader are documented in [docs/DAILY_NEWSPAPER.md](docs/DAILY_NEWSPAPER.md).
